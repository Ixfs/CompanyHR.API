API для учёта сотрудников компании. Разработка BackEnd идёт на Java, FrontEnd разрабатывается на Flutter (может быть изменён язык, заменить строку с языком для FrontEnd, если произойдёт смена языка).

Авторизация (прибытие и отход) на месте работает через скан случайного QR-кода.

Из кода удалить ненужный функционал (Кеширование, статистика). Упростить структуру. Ориентир - разработка именно мобильного приложения.
